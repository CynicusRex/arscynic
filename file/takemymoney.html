What if I paid for all my free software?

I go out of my way to find the best free software that doesn't rely on advertising. However, I've always felt guilty for not showing appreciation towards this rare breed of excellent humans. How much would I “lose” anyway? Let's find out.

Requirements for consideration.

  1. No advertising, trackers, etc., whatsoever.
  2. Free. All of the main features must be available for free.
  3. No already rich people/companies.
  4. No association with or reference to crypto“currencies” because these are greed incentivizing pyramid schemes.

Donation amount calculation.

The donation amount is in proportion to my usage and whether software is in constant development, e.g., an operating system (OS).

Mandatory mentions.
